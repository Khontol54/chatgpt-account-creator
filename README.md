# ChatGPT Account Creator Bot

Automated ChatGPT account creation bot with support for batch processing, email verification, and credential management.

## Features

✨ **Core Features:**
- Automated ChatGPT account signup
- Single and batch account creation
- Email verification automation
- Account credential storage and management
- Proxy support for distributed operations
- Rate limiting to avoid detection
- Comprehensive logging and error handling

🔧 **Technical Features:**
- Browser automation with Puppeteer
- SQLite database for account storage
- Multiple email provider support (Gmail, temp email, custom SMTP)
- Express.js REST API
- Configuration management via environment variables
- Winston logging system
